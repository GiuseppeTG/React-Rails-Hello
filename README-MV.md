# React-rails-hello

> A simple repo for testing how to run React in a Ror project


## Built With

- Ruby on rails
- React
- Webpack

## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites

Rails 7
Ruby v 3.1.2

### Setup

- Clone this repo and open it locally

### Install

- npm i
- bundle i

### Usage
